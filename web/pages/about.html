Uncovering HTTP with Custom Server

As the foundation of the web, HTTP is used to transfer data between servers and browsers, allowing users to access internet resources. Among the numerous requests handled independently, using URIs (addresses) to locate resources such as pages, images, and files, HTTP has its own methods, each with a specific purpose. This protocol manages requests sent by websites, for example, and the responses from servers.

GET

What is it?

GET retrieves information from a resource on the server. This entire website is only displayed to you because the browser made GET requests for these pages to the server!

POST

What is it?

POST is used to send information to the server, such as forms and file uploads.

DELETE

What is it?

DELETE instructs the server to remove resources from the server.
